How To Use The Equipment Scheduler

  1. Fill out the form at the bottom.
  2. Click the "+" sign to add the job to the screen.
  3. Drag the job onto the day you'd like to schedule it for.
  4. Click to copy the job into the form and re-add it to the screen. Feel free to make edits to the form and delete the old job accordingly by dragging it to the trash can.
  5. The "-" button will clear the form inputs so you can start fresh.
  6. Click the labels on the left-hand side to edit the names of the machines.
  7. If you'd like to schedule a job for a later date, use the "Change Date" button and follow the first 3 steps.
  8. Click "Save All" at the top when you're happy with how the schedule looks and follow the prompts to save the XML file to your computer.
  9. Load in a previously saved schedule using the "Load Schedule" button.
  10. Add a print note with the "Add Print Note" button. It will only display when you go to print the schedule using the "Print Schedule" button.
  11. If you'd like to make edits to the note, click "Add Print Note" again to edit the note.
  12. Search for a Job Number using the "Search for a Job#" button. Type in the job you're searching for and press go. It will display the date(s) for when that job is scheduled.
